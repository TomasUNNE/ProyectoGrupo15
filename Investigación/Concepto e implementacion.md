Manejo de Transacciones y Transacciones Anidadas en la Base de Datos “Caso_gimnasio”
El modelo de datos del Caso_gimnasio representa un sistema de gestión integral que está compuesta por entidades como Persona, Alumno, Membresía, Pago, Plan y 
Personal, entre otras. En este contexto, el manejo correcto de las transacciones resulta fundamental para garantizar la integridad, coherencia y confiabilidad de la 
información almacenada, especialmente ante escenarios donde se realizan operaciones simultáneas o dependientes entre múltiples tablas.
